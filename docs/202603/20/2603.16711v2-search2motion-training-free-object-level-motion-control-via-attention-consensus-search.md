## 摘要
我们提出了 Search2Motion，这是一个用于图像生成视频中对象级运动编辑的无需训练的框架。与以往需要轨迹、边界框、掩码或运动场的方法不同，Search2Motion 采用基于目标帧的控制，利用首尾帧运动先验来实现对象重定位，同时在无需微调的情况下保持场景稳定性。通过语义引导的对象插入和鲁棒的背景修复，实现了可靠的目标帧构建。我们进一步表明，早期步骤的自注意力图可以预测对象和相机的动态，提供可解释的用户反馈，并启发了 ACE-Seed（用于早期种子选择的注意力共识），这是一种轻量级搜索策略，可在无需前瞻采样或外部评估器的情况下提高运动保真度。注意到现有基准测试混淆了对象和相机的运动，我们引入了 S2M-DAVIS 和 S2M-OMB 用于稳定相机下的纯对象评估，以及 FLF2V-obj 指标，该指标在不需要地面真值轨迹的情况下隔离对象伪影。Search2Motion 在 FLF2V-obj 和 VBench 上始终优于基准模型。

## Abstract
We present Search2Motion, a training-free framework for object-level motion editing in image-to-video generation. Unlike prior methods requiring trajectories, bounding boxes, masks, or motion fields, Search2Motion adopts target-frame-based control, leveraging first-last-frame motion priors to realize object relocation while preserving scene stability without fine-tuning. Reliable target-frame construction is achieved through semantic-guided object insertion and robust background inpainting. We further show that early-step self-attention maps predict object and camera dynamics, offering interpretable user feedback and motivating ACE-Seed (Attention Consensus for Early-step Seed selection), a lightweight search strategy that improves motion fidelity without look-ahead sampling or external evaluators. Noting that existing benchmarks conflate object and camera motion, we introduce S2M-DAVIS and S2M-OMB for stable-camera, object-only evaluation, alongside FLF2V-obj metrics that isolate object artifacts without requiring ground-truth trajectories. Search2Motion consistently outperforms baselines on FLF2V-obj and VBench.

#### 1. 核心问题与研究动机
在图像到视频（I2V）生成领域，实现精确的对象级运动编辑（如让图中某个物体移动到特定位置）一直是个挑战。现有方法通常存在以下问题：
*   **输入复杂：** 需要用户提供精确的轨迹、深度图或复杂的掩码，这对普通用户门槛过高。
*   **耦合性强：** 往往需要针对特定模型进行微调或添加辅助控制模块，难以迁移到快速迭代的新模型上。
*   **质量退化：** 引入额外的控制信号有时会破坏预训练模型的原生生成能力，导致视频质量下降。
*   **评估混淆：** 现有基准测试往往将物体运动与相机运动混淆，难以独立评估物体运动的保真度。

**Search2Motion** 的核心动机是：利用预训练模型中已有的运动先验，通过“首尾帧”控制逻辑，实现无需训练、用户友好且高质量的物体运动编辑。

#### 2. 方法论
该框架由三个独立且可模块化的阶段组成：

*   **目标帧构建（Target Frame Construction）：**
    *   **语义引导放置：** 利用视觉语言模型（VLM，如 Qwen2.5-VL）建议合理的物体放置区域（如“在公路上”），并结合 SAM2 生成空间掩码。
    *   **合成与修复：** 使用 ObjectClear 等模型进行背景修复（移除原位置物体），将物体按用户意图（平移、旋转、缩放）粘贴到新位置，最后通过图像编辑模型（如 Flux-Kontext）进行融合，生成高质量的“最后一帧”。
*   **运动合成（Motion Synthesis）：**
    *   将任务转化为“首尾帧到视频”（FLF2V）生成任务。利用预训练的视频扩散模型（如 Wan2.2 或 VACE），以第一帧和合成的目标帧为条件生成中间过程。
*   **ACE-Seed（注意力共识种子选择）：**
    *   **核心思想：** 扩散模型在去噪的早期步骤（如 50 步中的前 10 步）就已经确定了宏观的运动结构。
    *   **算法流程：** 采样 $N$ 个随机种子，运行少量去噪步数，提取特定 Transformer 层的自注意力图。计算这些注意力图的“共识”（即与群体平均模式最接近的种子），认为这些种子产生的运动轨迹最稳健、最不容易出现伪影。

#### 3. 实验设计
*   **数据集与基准（Benchmarks）：**
    *   **S2M-DAVIS：** 基于 DAVIS2017 视频序列构建，通过合成手段确保背景静止，仅物体运动。
    *   **S2M-OMB：** 基于 ObjMove-B 数据集构建。
*   **评估指标：**
    *   **VBench：** 评估整体视频质量（一致性、平滑度等）。
    *   **FLF2V-obj（新提出）：** 利用 SAM2 跟踪生成的物体，计算物体裁剪图之间的 DINOv2 相似度和 LPIPS 距离，专门衡量物体本身的保真度。
*   **对比方法：** 与 DragAnything（基于轨迹控制）和 TTM（基于时间扭曲）等最先进的运动控制方法进行对比。

#### 6. 主要结论与发现
*   **无需微调即可控制：** 预训练的 FLF2V 模型内部已经蕴含了丰富的物体运动知识，关键在于如何通过高质量的目标帧和种子搜索将其激发出来。
*   **注意力图的预测性：** 扩散模型早期的自注意力图（尤其是中深层）能够有效预测最终视频的运动质量。
*   **共识即鲁棒：** 选择符合“注意力共识”的种子可以显著减少物体漂移、变形等常见扩散模型伪影。

#### 7. 优点
*   **零门槛：** 用户只需指定“物体去哪儿”，无需画复杂的轨迹。
*   **模型无关：** 作为一个插件式的管道，可以轻松适配任何新的 FLF2V 视频生成模型。
*   **可解释性：** 通过早期注意力图提供反馈，让用户在生成完成前就能预览运动趋势。
*   **评估创新：** 提出的 FLF2V-obj 指标填补了物体级运动评估的空白。

#### 8. 不足与局限
*   **依赖修复质量：** 目标帧的构建高度依赖背景修复模型的效果，如果修复留有残影，会直接影响视频质量。
*   **复杂交互限制：** 目前主要针对单物体的平移/旋转/缩放，对于多个物体之间复杂的物理交互（如碰撞、遮挡）处理能力尚待验证。
*   **推理延迟：** 虽然比全采样快，但 ACE-Seed 仍需对多个种子进行初步去噪，在实时性要求极高的场景下可能存在延迟。
