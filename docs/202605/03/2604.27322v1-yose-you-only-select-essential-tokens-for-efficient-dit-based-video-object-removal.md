## 摘要
最近，基于扩散 Transformer (DiT) 的视频生成技术在视频物体移除方面取得了令人瞩目的成果。然而，这些方法仍面临显著的推理延迟问题。例如，尽管 MiniMax Remover 达到了最先进的视觉质量，但其运行速度仅为 10FPS 左右，这主要是因为即使只有很小的掩码区域需要处理，它也会在整个时空 Token 空间上进行密集计算。在本文中，我们提出了 YOSE（You Only Select Essential Tokens），一个高效的微调框架。YOSE 引入了两个关键组件：批量变长索引 (BVI) 和扩散过程模拟器 (DiffSim) 模块。BVI 是一种可微的动态索引算子，能够根据掩码信息自适应地选择关键 Token，从而实现跨样本的变长 Token 处理。DiffSim 为未掩码 Token 提供了一种扩散过程近似机制，通过模拟 DiT 自注意力机制中未掩码区域的影响，来保持掩码 Token 的语义一致性。通过这些设计，YOSE 实现了掩码感知加速，其推理时间与掩码区域大小近似成线性比例，而全 Token 扩散方法的计算量则无论掩码大小如何都保持不变。大量实验表明，YOSE 在 70% 的情况下实现了高达 2.5 倍的加速，同时保持了与基准模型相当的视觉质量。代码可在以下网址获取：https://github.com/Wucy0519/YOSE-CVPR26。

## Abstract
Recent advances in Diffusion Transformer (DiT)-based video generation technologies have shown impressive results for video object removal. However, these methods still suffer from substantial inference latency. For instance, although MiniMax Remover achieves state-of-the-art visual quality, it operates at only around 10FPS, primarily due to dense computations over the entire spatiotemporal token space, even when only a small masked region actually requires processing. In this paper, we present YOSE, You Only Select Essential Tokens, an efficient fine-tuning framework. YOSE introduces two key components: Batch Variable-length Indexing (BVI) and Diffusion Process Simulator (DiffSim) Module. BVI is a differentiable dynamic indexing operator that adaptively selects essential tokens based on mask information, enabling variable-length token processing across samples. DiffSim provides a diffusion process approximation mechanism for unmasked tokens, which simulates the influence of unmasked regions within DiT self-attention to maintain semantic consistency for masked tokens. With these designs, YOSE achieves mask-aware acceleration, where the inference time scales approximately linearly with the masked regions, in contrast to full-token diffusion methods whose computation remains constant regardless of the mask size. Extensive experiments demonstrate that YOSE achieves up to 2.5X speedup in 70% of cases while maintaining visual quality comparable to the baseline. Code is available at: https://github.com/Wucy0519/YOSE-CVPR26.

#### 1. 核心问题与整体含义（研究动机和背景）
视频物体移除（Video Object Removal, VOR）旨在消除视频中不需要的物体并填补背景。目前基于扩散 Transformer（DiT）的方法（如 MiniMax Remover）虽然视觉质量极佳，但面临严重的**推理延迟**问题。
*   **研究动机**：DiT 模型通常对全视频时空 Token 进行密集计算。然而，在 VOR 任务中，通常只有局部区域（掩码区域）需要重新生成，约 70% 的实际场景中掩码占比低于 20%。
*   **核心问题**：如何避免对未掩码区域的冗余计算，使推理复杂度与掩码大小成线性关系，同时不损失视频的语义连贯性和视觉质量。

#### 2. 方法论：核心思想与技术细节
YOSE 提出了一个轻量级的微调框架，核心思想是“**只处理必要的 Token**”。
*   **批量变长索引（Batch Variable-length Indexing, BVI）**：
    *   这是一种可微的动态索引算子。它根据掩码信息提取关键 Token，并利用 `grid_sample` 函数实现连续插值采样，从而保证梯度回传。
    *   它解决了批处理中不同样本掩码大小不一的问题，通过动态填充实现高效的并行计算。
*   **扩散过程模拟器（Diffusion Process Simulator, DiffSim）**：
    *   为了在不计算全量 Token 的情况下维持语义一致性，DiffSim 模拟了未掩码区域在 DiT 自注意力机制中的影响。
    *   它引入了可学习的参数（组合参数 $G$、缩放参数 $S$、偏置 $Bias$），利用噪声潜变量和原始视频潜变量的残差来生成代理键（Key）和值（Value），使掩码 Token 能够感知全局上下文。
*   **融合策略（Fusion Strategy）**：
    *   在边界处进行均值-方差对齐（Mean-Variance Alignment），并使用加权掩码进行平滑混合，以消除重建区域与原始区域之间的视觉接缝。

#### 7. 优点（亮点）
*   **即插即用**：作为一个微调框架，它不需要重新设计复杂的 DiT 架构，仅需极少量的可学习参数即可适配现有模型。
*   **掩码感知计算**：打破了 DiT 必须处理全量 Token 的固有范式，极大地提升了局部编辑任务的实用性。
*   **背景保护**：天然地解决了生成模型在物体移除时容易误伤非目标区域的问题。

#### 8. 不足与局限
*   **极端场景限制**：当掩码区域覆盖全图（如 $\gamma > 80\%$）时，由于 VAE 的块编码约束，加速效果会消失，计算量退化至与原模型持平。
*   **依赖掩码质量**：该方法高度依赖输入掩码的准确性，若掩码未能完全覆盖物体及其副作用（如阴影），DiffSim 模拟的上下文可能会出现偏差。
*   **VAE 伪影**：虽然有融合策略，但在某些极端光影变化的视频中，3D-VAE 的因果编码特性仍可能在边界产生轻微的统计不一致。
