## 摘要
我们提出了 Search2Motion，这是一个用于图像到视频生成中对象级运动编辑的无需训练的框架。与以往需要轨迹、边界框、掩码或运动场的方法不同，Search2Motion 采用基于目标帧的控制，利用首尾帧运动先验来实现对象重定位，同时在无需微调的情况下保持场景稳定性。通过语义引导的对象插入和鲁棒的背景修复，实现了可靠的目标帧构建。我们进一步表明，早期步骤的自注意力图可以预测对象和相机的动态，提供可解释的用户反馈，并启发了 ACE-Seed（早期种子选择的注意力共识），这是一种轻量级搜索策略，可在无需前瞻采样或外部评估器的情况下提高运动保真度。注意到现有基准测试混淆了对象和相机的运动，我们引入了 S2M-DAVIS 和 S2M-OMB 用于稳定相机下的纯对象评估，以及 FLF2V-obj 指标，该指标无需地面真值轨迹即可隔离对象伪影。Search2Motion 在 FLF2V-obj 和 VBench 上始终优于基准方法。

## Abstract
We present Search2Motion, a training-free framework for object-level motion editing in image-to-video generation. Unlike prior methods requiring trajectories, bounding boxes, masks, or motion fields, Search2Motion adopts target-frame-based control, leveraging first-last-frame motion priors to realize object relocation while preserving scene stability without fine-tuning. Reliable target-frame construction is achieved through semantic-guided object insertion and robust background inpainting. We further show that early-step self-attention maps predict object and camera dynamics, offering interpretable user feedback and motivating ACE-Seed (Attention Consensus for Early-step Seed selection), a lightweight search strategy that improves motion fidelity without look-ahead sampling or external evaluators. Noting that existing benchmarks conflate object and camera motion, we introduce S2M-DAVIS and S2M-OMB for stable-camera, object-only evaluation, alongside FLF2V-obj metrics that isolate object artifacts without requiring ground-truth trajectories. Search2Motion consistently outperforms baselines on FLF2V-obj and VBench.

### 2. 方法论：核心思想与技术细节
Search2Motion 将物体运动编辑重新定义为**首尾帧生成（FLF2V）任务**，主要包含三个阶段：

*   **目标帧构建（Target Frame Construction）**：
    *   **语义引导放置**：利用视觉语言模型（VLM，如 Qwen2.5-VL）识别场景中合理的放置区域（如“天空”、“路面”），并结合 SAM2 生成空间掩码。
    *   **鲁棒背景修复与合成**：使用 ObjectClear 等模型对原图进行背景修复（Inpainting），将目标物体通过仿射变换（平移、旋转、缩放）放置到新位置，并利用图像编辑模型（如 Flux-Kontext）进行光影和谐化处理。
*   **运动合成（Motion Synthesis）**：
    *   将原始图像作为首帧，合成的图像作为末帧，输入预训练的 FLF2V 模型（如 Wan2.2 或 VACE）生成中间运动过程。
*   **ACE-Seed（注意力共识种子选择策略）**：
    *   **核心发现**：扩散模型在去噪的早期步骤（如前 10 步）中，其自注意力图（Self-Attention Maps）已经预示了物体的运动轨迹和相机动态。
    *   **算法流程**：
        1.  并行运行 $N$ 个随机噪声种子，仅执行前 $t_{early}$ 步去噪。
        2.  提取特定层（研究发现第 22-26 层最有效）的自注意力图。
        3.  **共识计算**：计算每个种子的注意力特征与群体平均特征的相似度（余弦相似度）。
        4.  **选择**：选择相似度最高的种子进行完整的视频生成。该策略基于“共识即鲁棒”的假设，旨在剔除产生运动伪影或背景漂移的异常种子。

### 6. 主要结论与发现
*   **FLF2V 范式的有效性**：通过构建高质量的末帧，可以比直接给定轨迹更有效地引导模型生成合理的物体运动。
*   **注意力图的预测性**：扩散模型的中间层注意力图是运动质量的早期指标，能够有效区分“好”和“坏”的生成种子。
*   **ACE-Seed 的优越性**：在不增加显著计算负担的情况下，ACE-Seed 显著提升了运动的保真度和场景稳定性，且与人类审美高度一致。

### 7. 优点与亮点
*   **无需训练且模块化**：可以轻松适配任何新的 FLF2V 基础模型，具有极强的通用性。
*   **可解释性**：利用注意力图作为反馈，为黑盒扩散过程提供了一定的可解释性。
*   **评估创新**：提出的 FLF2V-obj 指标解决了现有视频评估中“前景背景混淆”的问题，对物体级运动研究有重要参考价值。
*   **用户友好**：将复杂的轨迹设计简化为简单的“剪切-粘贴”逻辑。

### 8. 不足与局限
*   **依赖基础模型**：生成的上限受限于底层 FLF2V 模型的能力，如果基础模型无法处理复杂的非刚性变形，该框架也难以解决。
*   **背景修复瓶颈**：目标帧的质量高度依赖于背景修复（Inpainting）的效果，若修复出现伪影，会直接传播到视频中。
*   **运动类型限制**：目前主要针对物体重定位（Relocation），对于更复杂的交互式运动或大幅度的形态改变，单纯依靠首尾帧可能不足以精确描述。
